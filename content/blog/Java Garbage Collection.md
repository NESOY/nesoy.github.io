# Java Garbage Collection
## Garbage Collector

### GC Log Rotate
- `-XX:+UseGCLogFileRotation`
    - default : false
- `-XX:NumberOfGCLogFiles`
    - default : 0
- `-XX:GCLogFileSize`
    - default : 0
