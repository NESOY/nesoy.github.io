# Ghostty Font
- 모든 font에 대해 CJK(China, Japan, Korea)가 지원되지 않는다. 
- 그렇기에 CJK를 지원하는 fallback font-family를 하나 더 지정해야 한다.

## 예시
```
font-family = "MesloLGS NF"
font-family = "Noto Sans CJK KR"
```
